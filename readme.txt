== Description ==

Raiden is a stylish, flexible and minimal WordPress theme for publishers. Comes packed with 6 color schemes, still allowing for complete customization over colors. The theme also includes 3 different layout options for your blog page / homepage so you can choose the one that suits you best.

Raiden is not just Responsive, it is mobile-optimized, ensuring that your content will look gorgeous on any viewport, regardless of the device.

== Installation ==

1. Download theme zip.
2. From your wp-admin Dashboard, go to Themes >> Add New and Upload theme.
3. Once installed, activate the theme.
